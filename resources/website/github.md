# github

## 自定义域名

> 在GitHub上, 导航至你的某个网页项目仓库（GitHub Pages site's repository）。 **在“Custom domain,”下输入框中添加你的自定义域名（你之前购买的域名想要指向该项目作为网站，如： jaylin.top ），之后点击Save.按钮保存**。
