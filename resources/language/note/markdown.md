## Markdown与思维导图

## Markdown 中如何实现内容折叠操作

使用 html 代码可以实现内容折叠的效果

```html
<details>
<summary>展开查看</summary>
<pre><code>
System.out.println("Hello to see U!");
</code></pre>
</details>
```

### 效果

点击三角形或者文字 --展开查看-- 都会触发展开收起
